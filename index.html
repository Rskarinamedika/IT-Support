Backend IT Jobdesk

  • ⏺ Pengembangan SIMRS: Membangun dan memperbarui modul Sistem Informasi Manajemen Rumah Sakit (SIMRS) seperti pendaftaran, kasir, farmasi, dan rawat inap agar terintegrasi secara real-time
  • ⏺ Manajemen Basis Data: Merancang arsitektur dan mengelola basis data yang menyimpan riwayat medis pasien secara rapi, cepat, dan aman.
  • ⏺ Integrasi API Eksternal: Menghubungkan sistem rumah sakit dengan pihak eksternal, seperti integrasi billing dengan BPJS Kesehatan, BPK RI, dan portal SATUSEHAT milik Kementerian Kesehatan.
  • ⏺ Keamanan Data Klinis: Menerapkan enkripsi data yang kuat untuk melindungi kerahasiaan rekam medis pasien sesuai dengan regulasi privasi data kesehatan.
  • ⏺ Optimalisasi API & Server: Memastikan Application Programming Interface (API) bekerja optimal sehingga komunikasi data antara aplikasi front-end (suster/dokter) dan server berjalan cepat tanpa downtime.
  • ⏺ Monitoring Mikrotik Server Provider (Biznet & Indihome)
